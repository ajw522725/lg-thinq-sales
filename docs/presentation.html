외부 맥락 미반영

🔢

우선순위 판단 주관적

📋

전략 보고 수작업

🏆

Solution

ThinQ-Sales 핵심 3가지

수집부터 인사이트까지 하나의 파이프라인으로 — VOC 20건을 넣으면 리드 스코어와 전략 추천이 자동으로 나옵니다.
01

멀티 플랫폼 VOC 통합 수집

Danawa · Reddit · 네이버 블로그 · YouTube · X/Twitter를 단일 파이프라인으로 수집. 텍스트 정규화·중복 제거 자동화.

02

AI 기반 리드 스코어링

감성·구매의도·긴급도·경쟁사 언급·외부 맥락을 종합한 0–100 리드 점수. 영업 우선순위를 데이터로 결정.

03

LLM 전략 인사이트 자동 생성

VOC·리드 점수·외부 컨텍스트를 조합해 "지금 이 고객에게 무엇을 제안해야 하는가"를 자동 생성.
